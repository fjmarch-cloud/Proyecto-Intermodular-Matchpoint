# Análisis de datos – MatchPoint

## Tipo de empresa
MatchPoint es una plataforma de gestión de reservas de pistas deportivas.

## Información que gestiona
- Usuarios registrados
- Pistas deportivas
- Reservas de pistas
- Pagos asociados a reservas

## Entidades identificadas
- Usuarios
- Pistas
- Reservas
- Pagos

## Relaciones
- Un usuario puede realizar varias reservas
- Una pista puede tener varias reservas
- Cada reserva pertenece a un usuario
- Cada reserva está asociada a una pista
- Cada reserva puede tener un pago asociado
