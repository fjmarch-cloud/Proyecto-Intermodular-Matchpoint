# Presentación del proyecto – MatchPoint

## ¿Qué es MatchPoint?

MatchPoint es una aplicación orientada a la gestión de reservas de pistas deportivas, como tenis, pádel o fútbol.

## Problema que resuelve

Muchos centros deportivos gestionan reservas de forma manual o poco eficiente. Esta aplicación permite organizar reservas de manera estructurada, evitando conflictos de horarios y facilitando la gestión de usuarios y pistas.

## ¿A quién va dirigido?

- Usuarios que quieren reservar pistas deportivas
- Centros deportivos que necesitan organizar sus reservas

## Funcionalidades principales

- Gestión de usuarios
- Consulta de pistas disponibles
- Creación de reservas
- Cancelación de reservas
- Consulta de reservas

## Tecnologías utilizadas

- HTML y CSS (interfaz web)
- Java (lógica de aplicación)
- MySQL (base de datos)
- JDBC (conexión con la base de datos)
- GitHub (control de versiones)

- Conectar aplicaciones Java a bases de datos mediante JDBC
- Implementar operaciones CRUD
