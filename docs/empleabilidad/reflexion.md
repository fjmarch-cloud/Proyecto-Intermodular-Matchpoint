# Reflexión personal

Este proyecto me ha ayudado a entender cómo se conectan los diferentes módulos del ciclo DAW en un caso real.

Lo que más me ha costado ha sido organizar correctamente la base de datos y entender bien la relación entre tablas.

También he aprendido la importancia de trabajar con GitHub de forma ordenada y hacer commits progresivos.

Si tuviera más tiempo, mejoraría la interfaz de usuario y añadiría más funcionalidades.

En general, este proyecto me ha servido para acercarme a un entorno de desarrollo más realista y entender mejor el trabajo de un desarrollador web.
