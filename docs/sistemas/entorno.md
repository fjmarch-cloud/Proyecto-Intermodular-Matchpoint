# Entorno de desarrollo - MatchPoint

## 1. Sistema operativo
El proyecto ha sido desarrollado en Windows 10/11.

## 2. Lenguajes utilizados
- Java (JDBC para conexión con base de datos)
- HTML5
- CSS3
- SQL (MySQL)

## 3. Entorno de desarrollo (IDE)
- Visual Studio Code para frontend
- IntelliJ IDEA para Java

## 4. Servidor de base de datos
- MySQL Server 8.x
- MySQL Workbench para gestión de BD

## 5. Conector utilizado
- mysql-connector-j-9.6.0

## 6. Arquitectura del sistema
El sistema se basa en una arquitectura cliente-servidor:

- Cliente: interfaz web (HTML/CSS)
- Servidor: lógica en Java con JDBC
- Base de datos: MySQL

## 7. Funcionamiento general
El usuario realiza reservas desde la interfaz web, que son procesadas por Java y almacenadas en la base de datos.
