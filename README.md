# MatchPoint 🎾

## Descripción
MatchPoint es una aplicación web orientada a la gestión de reservas de pistas deportivas.

## Funcionalidades
- Gestión de usuarios
- Consulta de pistas
- Creación y cancelación de reservas

## Tecnologías
- HTML / CSS
- Java
- MySQL
- JDBC

## Estructura
/web → interfaz web  
/src → código Java  
/sql → scripts base de datos  

## Ejecución
1. Importar base de datos
2. Ejecutar aplicación Java
